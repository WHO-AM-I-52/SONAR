## 💡 Идеи

### v2.4 — OCR и автозаполнение
- Автозаполнение полей формы по скану анкеты (PDF/DOCX)
- Поддержка форматов: PDF, DOCX, JPG, PNG
- Просмотр результатов распознавания перед сохранением

### v2.5 — Уведомления на почту
- Уведомления на почту при назначении обращения
- Напоминания о просроченных обращениях
- Настройка уведомлений в профиле пользователя

### Автообновление в интерфейсе
- Встроенная проверка новых версий при запуске
- Автоматическое обновление кода из GitHub одной кнопкой
- Отображение текущей и доступной версии в интерфейсе

### v3.0 — Автоматический подбор земельных участков
- Таблица земельных площадок НО (land_plots) в БД
- Алгоритм подбора участков по параметрам заявителя (район, ОКВЭД, площадь)
- Блок «Рекомендуемые участки» в карточке обращения (топ-5)
- Привязка участка к обращению одним кликом
- История привязок в activity_log

### v3.1 — ИИ-поиск по участкам (Perplexity Sonar API)
- Поиск участков по вопросу на естественном языке
- Интеграция с Perplexity Sonar API
- Поиск по внутренней базе land_plots с передачей контекста
- Flask-маршрут /api/ai-land-search

### v3.2 — Telegram-уведомления (бот-отправщик)
- Односторонний канал SONAR → Telegram
- Уведомления о новых, принятых, возвращённых, отвеченных обращениях
- Ежедневная сводка из dashboard по расписанию
- Уведомление о новых релизах

### v3.3 — Telegram как канал приёма заявок
- Двусторонний канал: пользователь пишет боту → заявка в SONAR
- FSM-анкета через aiogram 3.x
- Заявка сохраняется с source_type='Telegram'
- Фильтрация по источнику в интерфейсе SONAR
