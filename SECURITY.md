# Политика безопасности SONAR

## Поддерживаемые версии

Текущая активная ветка: `main`.

## Как сообщить об уязвимости

Пожалуйста, **не создавайте публичный Issue** для уязвимостей.

Сообщайте через GitHub [Private vulnerability reporting](
https://github.com/WHO-AM-I-52/SONAR/security/advisories/new)
или свяжитесь с владельцем напрямую.

## Что включать в отчёт

- Описание уязвимости
- Шаги для воспроизведения
- Версия SONAR, в которой обнаружена проблема
- Скриншот (если возможно)
